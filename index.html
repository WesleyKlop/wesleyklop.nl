About me

Hi! My name is Wesley Klop and I am a Dutch web development student at the Grafisch Lyceum Rotterdam. I'm currently 18 years old and I have a passion for making websites and working with servers. (Like the one hosting this website!)

My interest for computers was sparked in the first class of middle school when I was around 13-14 years old. When I had IT class the teacher would always ask me if I could help the rest of my classmates with their issues! It was that same year I wrote my first .html file using notepad.

Since then I have learned a lot about the whole process of making a website and getting it running. I'm very interested in both making websites and making sure they are running on an up-to-date and stable server.

Projects

These are some of the projects I have worked on since about a year or two ago, with my favorite projects on top!

RatchetChat

This is an online chat system using web sockets with the PHP library Ratchet as the backend. It also uses an LDAP library to authenticate with the domain controller to let domain users login with their existing account. It will support banning users, logging the chat and loading older messages. I am still working on this projects for a school assignment. Check it out on Github!

Things learned: PHP, Ratchet, Composer, Git, PhpStorm, ECMAScript 6, Material Design Lite.

Travel Tips

This was a school project in which I made a mobile app that would present the user with a list of countries with every country containing tips and recommendations when visiting. Most of my classmates made this using html/css/js combined with Phonegap but I decided to try and do it natively in Java. It was very hard for me but in the end it turned out very well with things like Google Sign In and a handwritten PHP backend that served JSON.

In this project I learned a lot about Object Oriented Programming, Java, JSON and creating API's with PHP.

Things learned: PHP, Composer, API's, Java, Android Studio, JSON, Volley, Object Oriented Programming.

Sinterklaas Lootjes

This was an interesting website I made together with my classmate Sam van der Form. On this website I was responsible for most of the backend. This was also my first time applying OOP with PHP, and using Git and Grunt on a project. I think it turned out quite well. Check it out on Bitbucket!

Things learned: PHP, HTML5, CSS3, Bootstrap, Git, Grunt, Bower, NPM, Vim, Object Oriented Programming.
